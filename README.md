# ToMystbin

Tiny Discord bot helper to send code to Mystbin for easy sharing.

## Requirements

- Python 3.13+
- [uv](https://docs.astral.sh/uv/)

## Setup

1. Clone the repository:
```bash
git clone <repo-url>
cd ToMystbin
```
2. Rename `config.toml.example` to `config.toml`.
3. Fill in your bot token and any required config values.
4. Run the bot:
```bash
uv run main.py
```
